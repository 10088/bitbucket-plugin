Overrides the URL used for matching from "Repository URL" to the value placed here.

Self-hosted Bitbucket users may need to set this to '<project key>/<repo>' for example 'prj/example-repo'

To troubleshoot URL matching go see https://support.cloudbees.com/hc/en-us/articles/115000051112-Bitbucket-Webhooks-Troubleshooting
